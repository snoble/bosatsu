JIT Performance Benchmark

Map Traversal vs Generated Code - Measuring the potential speedup

What This Measures

This benchmark compares two approaches to updating DOM elements:

  1. Map Traversal (Current): Look up bindings in _bindings[path], iterate through them, find elements, apply updates
  2. Generated Code (JIT): Direct element.textContent = value with pre-cached element references

Exit Criterion: If simulated speedup is <1.5x at 100 bindings, the JIT approach may not be worth the complexity.
